What This Template Does

Fibo Edit – Add Object by Text inserts new objects into an existing image while maintaining perspective, lighting, and scene consistency.

How It Works

1️⃣ Upload Image

Select an image where you want to add a new object.

2️⃣ Describe the Object

Write what to add and where to place it.

3️⃣ Generate

The AI inserts the object naturally into the scene.
